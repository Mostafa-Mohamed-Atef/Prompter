# Prompter

Prompter is a Chrome extension that allows you to store, manage, and drag prompts into AI chat models, It streamlines your workflow by letting you save your favorite prompts and quickly send them to your chatbot with a single click.

## Features

- Save, edit, and delete your favorite prompts
- Quickly input saved prompt.
- Simple and clean popup interface
- Prompts are stored locally in your browser

## Installation

1. Clone or download this repository to your computer.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode" (top right).
4. Click "Load unpacked" and select the project directory.
5. The Prompter extension should now appear in your extensions bar.

## Usage

1. Navigate to any AI chat model.
2. Click the Prompter extension icon in your browser.
3. Add a new prompt using the input field and "Save" button.
4. select your prompt's text and drag and drop it in your chat box.
5. Use the ✏️ button to edit a prompt or 🗑️ to delete it.

## Permissions

- `storage`: To save your prompts locally.
- `activeTab`, `scripting`: To inject prompts into the ChatGPT page.
